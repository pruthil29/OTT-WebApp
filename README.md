# OTT Web Application

A Java-based OTT movie web application developed using Servlet, JSP, and MySQL.

## Technologies Used
- Java
- Servlet & JSP
- MySQL
- JDBC
- HTML
- CSS

## Features
- User Login
- Admin Login
- Add Movie
- Edit Movie
- Delete Movie
- Search Movies
- Watch Trailer (YouTube Embed)
